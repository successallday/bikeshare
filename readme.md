# Bikeshare Analysis

This project analyzes bikeshare trip data to understand differences between member and casual riders.

## Tools Used
- SQL (Google BigQuery)

## Key Analysis
- Combined monthly trip data into a single dataset
- Compared average ride duration between member and casual users
- Analyzed ride frequency by day of week
- Examined usage of different bike types

## Key Insight
Casual riders tend to have longer average ride durations, while members take more frequent rides.
